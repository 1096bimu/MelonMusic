# Melon Music 

I created this app because I was frustrated a seemingly simple combination of features does not seem to exist in any existing music player. Especially the following three features:
- Exposed documents folder for music files
- Sort music by Genre
- Ability to see the next song to be played
Somehow these things just don't exist in any single app, or at least not in one that's of reasonable size and complexity, the market is clearly dominated by streaming/cloud integration and fancy visuals.

When I decided to design this for my self, I started from the simple observation that I NEVER used the progress bar, and I HATE the back button on the traditional music player layout because it's always very mysterious what they're supposed to do, should they go back to the beginning? Should they go back to the previous song? Why would I need a button to go back to the beginning when there is already a progress bar? I just thought there must be a more elegant solution.

Melon Music is my answer, instead of progress bar, I have placed a click wheel that always shows the previous/next couple songs which you can easily access. I think this is what people actually do when listening to music, they pick tracks, they don't scrub around within the same track.
